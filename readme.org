#+TITLE: Readme

* cli-bindings

A clojure cli tool that supports tools meant for keybindings

** feats
*** Screenshot
*** Universal Focus Movement
*** Open X, where x is
- Workspace (with apps)
- Emacs
- Terminal
- Browser

* development
run it

#+BEGIN_SRC zsh
bb --classpath src --main cli.core -- -h
#+END_SRC

run an nrepl server to connect to.

#+BEGIN_SRC zsh
rlwrap bb --nrepl-server 1667
#+END_SRC
